## üêõ Reporting Bugs

Found a bug? Please report it using our [bug report template](.github/ISSUE_TEMPLATE/bug_report.yml):

1. Go to [Issues](https://github.com/Andreas-Garcia/audiometa-python/issues)
2. Click **New Issue**
3. Select **Bug Report**
4. Fill out the template with:
   - Description of the bug
   - Steps to reproduce
   - Expected vs actual behavior
   - Your environment (Python version, OS, AudioMeta version)
   - Code examples (if applicable)

## üìñ Common Questions

### Installation Issues

- **Problem**: Can't install AudioMeta
- **Solution**: Check [README.md Installation section](README.md#installation) for system requirements and external tool setup

### External Tools Required

AudioMeta requires external tools for full functionality:

- **ffprobe** (for WAV file processing)
- **flac** (for FLAC MD5 validation)

See [README.md System Requirements](README.md#system-requirements) for installation instructions.

### Getting Help Faster

To get help faster, please include:

- Your Python version (`python --version`)
- Your AudioMeta version
- Your operating system
- The audio file format you're working with
- Relevant code snippets
- Error messages (full traceback if available)
